# 📝 Gestor de Tareas en Java

## Autores
> Ariadna Izquierdo Méndez y Joel Peña Farías

## Descripción
Este proyecto es un programa de **gestión de tareas** desarrollado en **Java**, que permite a los usuarios añadir y listar tareas de manera sencilla. Está diseñado para ser fácil de usar y sirve como práctica para aprender a manejar clases, objetos y estructuras de datos en Java.

## Características

- Listar todas las tareas registradas.  
- Editar o eliminar tareas existentes.
  
## Tecnologías

- **Lenguaje:** Java 17 o superior  
- **IDE recomendado:** NetBeans o Intellij
